# Peer-to-peer Explicit Data Movement

## Description
Benchmark for bandwidth and latency of explicit data transfers between each pair of GPUs in the system, using hipMemcpy API.

## Build
`make`

## Run

```
./p2pBandwidthLatencyTest
```
